# Hidemymadness.py

- Extract Hidemyass.com proxy list (bypassing CSS obfuscation)
- Test proxies in 3 different ways
- Save result in csv files

Enjoy.

### Basic usage

```
python3 hidemymadness.py -e -t <url>
```
